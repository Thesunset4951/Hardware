# Hardware
เข็มขัดตรวจวัดการหกล้ม เพื่อช่วยเหลือผู้สูงอายุและผู้พิการ ประกอบด้วย เซนเซอร์ตรวจจับการหกล้ม(MPU6050)(ดูจากความเร่งในแกนx,y,zที่เปลี่ยนแปลงไป) อุปกรณ์ระบุตำแหน่งผู้ป่วยและอุปกรณ์แจ้งเตือน

บอร์ดที่ 1 : บอร์ดของผู้สูงอายุ เชื่อมต่อกับเซนเซอร์ตรวจจับการหกล้ม(MPU6050) ที่สามารถตรวจวัดความเร่งในแนวแกน x,y,z ที่เปลี่ยนไปอย่างฉับพลัน เพื่อระบุความรุนแรงของการหกล้ม โดยจะแยกระดับความรุนแรงจากช่วงค่าความเร่งที่เปลี่ยนไป และมีอุปกรณ์ติดตามตำแหน่งผู้ป่วย(GPS) และส่งข้อมูลไปยังบอร์ดที่ 2(บอร์ดของผู้ดูแล) ในขณะเดียวกัน จะส่งข้อมูลไปยังแอพลิเคชัน Google Chat เพื่อส่งการแจ้งเตือนไปยังโทรศัพท์ของผู้ดูแล โดยจะระบุตำแหน่งของผู้ป่วยตามพิกัดของ Google Maps และความรุนแรงของการหกล้มในหน้าการแจ้งเตือนของโทรศัพท์ เมื่อเสร็จสิ้นการส่งข้อมูล จะเปิดใช้งานเครื่องส่งเสียง(buzzer)และมีไฟสีแดง(red LED)กระพริบทันที ถ้าหากเซนเซอร์ตรวจจับการหกล้มรับข้อมูลผิดพลาด เช่น ทำอุปกรณ์ตรวจวัดการหกล้มตกพื้น แต่ผู้ป่วยไม่ได้ล้ม จะทำให้ buzzer ส่งเสียงดังและไฟ LED สีแดงกระพริบ ผู้ป่วยสามารถหยุดการทำงานได้โดยการกดสวิตช์บนบอร์ด

บอร์ดที่ 2 : บอร์ดของผู้ดูแล เป็นบอร์ดที่คอยรับข้อมูลจากบอร์ดที่ 1 เมื่อบอร์ดที่ 2 ได้รับข้อมูลการหกล้มจากบอร์ดที่ 1 จะมีการส่งเสียงแจ้งเตือนจาก buzzer ตามระดับความรุนแรงของการหกล้มด้วยเสียงสัญญาณที่แตกต่างกัน พร้อมกับไฟสีแดง(red LED)ที่กระพริบในทันที จากนั้นหน้าจอ OLED จะแสดงระดับความรุนแรงตามการหกล้มของผู้ป่วย ผู้ดูแลสามารถหยุดการทำงานของ buzzer และ LED สีแดงได้โดยการกดสวิตช์บนบอร์ด

แอพลิเคชัน Google Chat สำหรับแจ้งเตือนสถานะและตำแหน่งของผู้ป่วยไปยังโทรศัพท์ของผู้ดูแล
